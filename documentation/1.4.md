# Vii 1.4 —

> *Python syntax. C semantics. One language for everyone.*

---

## What is Vii?

Vii is the language that shouldn't exist.

It's interpreted **and** compiled.  
It's beginner-friendly **and** systems-level.  
It's safe by default **and** lets you touch raw memory when you need to.

Other languages force you to choose. Vii lets you grow.

---

## Write Your First Program (Beginner)

```vii
"Hello, World!"
```

That's it. No `main()`. No semicolons. No ceremony.

Get user input:
```vii
name = ask
"Hello, " + name 
```

Simple control flow:
```vii
if x > 10
  "big" 
else
  "small" 
```

Functions with implicit returns:
```vii
do add a b
  out a + b

result = add 5 3
```

**Feels like Python. Runs like C.**

## Same Language, Full Control (Expert)

Add types when you need performance:

```vii
# Explicit types, zero overhead
x i32 = 5
y f64 = 3.14159
arr [1024]u8 = stack.alloc    # 1KB on stack, no malloc
```

Fixed-size arrays with bounds checking in debug:
```vii
matrix [4][4]f32               # 64 bytes exactly, cache-friendly
val = matrix at 2 3            # bounds-checked in debug, raw in release
```

Structs with explicit layout:
```vii
struct Vec2
  x f32
  y f32                        # 8 bytes, no surprises

struct Packet packed           # packed, no padding
  id u16
  data [4]u8

sizeof Packet                  # 6, guaranteed
alignof Packet                 # 1, guaranteed
offsetof Packet data           # 2, guaranteed
```

## Memory: Your Choice, Visible Costs

### Stack (Automatic, Fast)
```vii
buf [4096]u8 = stack.alloc     # Lives until function returns
# No free needed - automatic cleanup
```

### Heap (Manual, Persistent)
```vii
ptr *u8 = heap.alloc 1024      # Raw allocation
*ptr = 42                      # Dereference
free ptr                       # Explicit cleanup
```

### Custom Allocators
```vii
arena "scratch" 16*1024*1024     # 16MB scratch space
buf = arena.alloc "scratch" 256
arena.reset "scratch"          # Bulk free, O(1)
```

**Every allocation strategy is one keyword away.**

## Unsafe: The Escape Hatch

Sometimes you need to break the rules. Vii makes it explicit:

```vii
# Safe by default - bounds checked
arr = [10]i32
val = arr at 5                 # runtime check

# Enter the danger zone
unsafe
  ptr = arr.ptr + 20           # pointer arithmetic
  val = *ptr                   # raw dereference
  mem.write ptr 69 i32         # raw write
end
```

**Unsafe is a block, not a dialect.**  
No rewriting your program. Just wrap the dangerous bits.

## Inline Assembly

Drop into assembly when you need it:

```vii
# Exit syscall on x86_64 Linux
asm
  mov rax, 60                  # sys_exit
  mov rdi, 0                   # status 0
  syscall
end
```

Or use memory-mapped I/O for embedded:
```vii
# Write to UART on bare metal
unsafe
  mem.write 0x09000000 0x41 u8  # 'A' to PL011 UART
end
```

## Systems Programming

### Freestanding Mode (No OS)
```bash
vii kernel.vii --freestanding -o kernel.bin
```

- No libc
- No runtime
- Custom `_start` entry
- Raw syscalls

### FFI: Call C Seamlessly
```vii
extern "printf" i32 (ptr u8, ...)  # Declare C function
extern "open" i32 (ptr u8, i32, ...)  # Varargs

fd = open "/etc/passwd" 0
printf "fd = %d\n" fd
```

**Use any C library. No wrappers. No bindings.**

## The Secret: Transparency

Vii transpiles to readable C. See exactly what runs:

```bash
$ vii program.vii --emit-c
```

```c
// Generated C - clean and predictable
int32_t x = 5;
float arr[4] = {1.0f, 2.0f, 3.0f, 4.0f};
struct Vec2 v = {.x = 10.0f, .y = 20.0f};
```

**Learn systems programming by seeing the translation.**

## REPL: Explore Dangerously

Interactive low-level programming:

```vii
$ vii
>>> ptr = heap.alloc 1024
>>> *ptr = 42
>>> *ptr
42
>>> free ptr
>>>
```

Test pointer logic interactively. Catch segfaults in the REPL, not production.

## Examples

### Binary File Parser
```vii
struct Header
  magic [4]u8
  size u32
  version u16

# Read raw bytes
file = "data.bin" ask
buf [256]u8 = stack-alloc
read file buf

# Parse at offset
hdr = buf at 0 as Header         # zero-copy cast
if (hdr.magic at 0) == 0x7F
  "Valid file" put
```

### Kernel Syscall
```vii
do sys_write fd ptr len
  out asm
    mov rax, 1                   # sys_write
    mov rdi, fd
    mov rsi, ptr
    mov rdx, len
    syscall
  

sys_write 1 "Hello\n" 6          # Write to stdout
```

### Memory-Mapped Graphics
```vii
unsafe
  fb = 0xB8000 as *u16           # VGA text buffer
  i = 0
  while i < 2000                 # 80x25
    *(fb + i) = 0x0F00 or 'A'    # white-on-black 'A'
    i = i + 1
  
end
```
